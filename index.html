Redux Roguelike Demo

A data-driven roguelike framework with Redux-style state management and unified input interface.

  • Arrow Keys - Move player / Navigate menus
  • Space or Enter - Action button / Select
  • I - Open inventory
  • G - Open gambling menu
  • R - Rest (restore HP/MP)
  • Esc - Return to world view
  • F1 - Toggle debug panel

All actions are dispatched through a Redux store with undo/redo support!
